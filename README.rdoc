== TODO list management

TODO list management allows to create and to manage ToDo lists and its subtasks.
Its also possible see public lists created by others users and favorite them.

```
ruby 2.2.3
rails 4.2

```

After clone this project:

```
$ bundle install
$ bundle exec rake db:setup
```

===== Run app
 `bundle exec rails s`

===== Run specs
 - Necessary install phantomjs to run features specs
 `sudo apt-get install phantomjs`
 `bundle exec rspec`
